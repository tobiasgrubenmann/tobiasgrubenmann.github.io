has environmentop back to ToC or Object Property ToC

IRI: https://w3id.org/timebank#hasEnvironment

The environment of a user or request. The environment of a request describes specific environmental factors of a request. The environment of a user can be used to infer environmental factors of a request if the user is the originator of the request. However, not all environmental factors of a user might apply to its request, especially if the location of the request differs from the users location.
has domain
Request c or User c
has range
Environment c
